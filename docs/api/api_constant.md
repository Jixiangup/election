# 异常CODE
|  状态码  |                   描述                   |
|:-----:|:--------------------------------------:|
|   0   |                  请求成功                  |
|  -1   |                  请求失败                  |
| 10000 |              客户端请求服务端授权失败              |
| 40000 | 客户端参数校验异常,会根据具体参数校验失败来回显给客户端,如: 身份证不合法 |
| 40001 |               邮箱或身份证已经存在               |
| 40002 |               候选人昵称已经存在                |
| 40003 |      不合法的状态操作, 如用户给已经`结束`的选举进行投票       |
| 40004 |          数据不存在, 如 `候选人不存在` 等           |
| 40005 |                  重复投票                  |
| 40006 |               不支持的分布式锁类型               |
| 40007 |                需要先参与投票                 |
| 50000 |                  系统异常                  |
